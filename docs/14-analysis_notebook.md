# Input format 
The QC pipeline is organized into two main chapters: cCRE-BC association, and RNA and DNA quantification.
Each chapter is accompanied by a dedicated script that allows the user to run all analyses presented in this book.

A detailed description of the required input file formats is provided in the GitHub pipeline repository: https://github.com/GokhmanLabOrganization/MPRA_QC_analysis/INPUT_FORMATS.MD.
